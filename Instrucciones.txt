1.1 Ventana para iniciar sesión
    - usuario
    - contraseña
    - opción de iniciar sesión/registrarse (solo tipo alumno)

1.2 Registro
    - Estudiante
        • Nombre
        • Apellido
        • DPI
        • Fecha de nacimiento
        • Teléfono
        • Nombre de usuario (deberá de ser único)
        • Dirección de correo electrónico
        • Contraseña
        • Confirmación de contraseña
    - Catedrático
        • Nombre
        • Apellido
        • DPI
        • Contraseña
        • Confirmación de contraseña
    - El usuario catedrático tendrá que ser registrado por el admin.

1.3 Recuperación de contraseña
    - Solicitar correo con el que se ha registrado

1.4 Bloqueo de cuenta
    - Si un usuario ingres de forma incorrecta su contraseña más de 3 veces se bloquea la cuenta de manera automática.
    - Se manda un correo al usuario indicando que su cuenta ha sido bloqueada.
       Argumento: Deberá solicitar al admin que la desbloqueee

2. Base de datos
    - usuario
    - contraseña
    - tipo de usuario
        - Admin
        - Cat.
        - Alumno



A. Comprobar si el usuario existe y el tipo de usuario.
Si no se posee cuenta tendrá que registrase
Si si se posee cuenta acceder a la vista según el tipo de usuario.

B. Cerrar sesión tras entrar.
